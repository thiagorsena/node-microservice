Example project for:

- Micro-service APIs
- Docker & Docker Compose
- NGINX
- Node 6.x
- TypeScript 2.0
- Express 4.0
- Socket.io Websockets
- PassportJS Auth
- Swagger 2.0
- Neo4j
- MongoDB


## Getting Started
### Pre-reqs
- [Docker](https://docs.docker.com/engine/installation/)
- [Docker Compose](https://docs.docker.com/compose/)

### Running
Install and start docker by `docker-compose up`.
